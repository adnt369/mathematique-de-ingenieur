Mathématiques de l'Ingénieur

Travaux Dirigés : Structures des Ensembles

Exercice 1

Soient les ensembles :

  • A = {x ∈ N : x est un multiple de 2}
  • B = {x ∈ N : x est un multiple de 3}
  • C = {x ∈ N : x est un multiple de 6}
  • D = {x ∈ N : x est un multiple de 8}

Déterminer :

  • A ∩ B
  • A ∩ C
  • A ∪ C
  • B ∪ C
  • C ∩ D

Exercice 2

Soit E = A ∪ N où A = {a, b, c, . . . , z} est l’alphabet latin. On pose :

  • A = {entiers naturels pairs inférieurs à 12}
  • B = {a, b, c, d}
  • C = {1, a, 3, b, 4, 8}

Réaliser un diagramme de Venn correspondant à A, B et C.

Calculer les ensembles suivants :

  • A ∩ C
  • B ∪ C
  • A ∪ B ∪ C

Exercice 3

En utilisant les quantificateurs, montrer que la réunion est distributive par rapport à l’intersection, c’est-à-dire :

A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

Exercice 4

Soient les ensembles A = {2, 4, 6, 8, 10}, B = {2, 3, 5, 7, 10} et C = {1, 2, 6, 7, 8, 9}.

Réaliser le diagramme de Venn.
